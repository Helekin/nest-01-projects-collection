## Overview

This repository is a practical review of TypeScript fundamentals. Each topic is implemented in a small, isolated example so you can study concepts without distractions.

## Topics Covered

- Basic and advanced **types**
  - primitives, unions, intersections, literal types
  - `unknown` vs `any`
  - type guards, type narrowing
  - generics
- **Objects** and structural typing
  - interfaces vs type aliases
  - optional properties, readonly
  - index signatures
- **Classes**
  - access modifiers (`public`, `private`, `protected`)
  - `readonly`
  - inheritance, abstract classes
  - getters/setters
  - static members
- **Dependency Injection (DI)**
  - constructor injection patterns
  - manual container vs simple service registry
  - interface-based design
- **Decorators**
  - class, method, accessor, property, parameter decorators
  - practical use cases (logging, validation, metadata)
- **Deprecated / Legacy Decorators**
  - understanding `experimentalDecorators`
  - differences between legacy and newer decorator proposals
  - migration considerations
